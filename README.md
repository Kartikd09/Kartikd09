### 🚀 Currently Building

**[FuseGuard](https://github.com/Kartikd09/Fuseguard)** — *the circuit breaker for AI agents.*
A drop-in proxy that **hard-blocks** the next LLM call before it breaches your budget — stopping runaway agent loops *before* the bill, not after. Open-core (MIT proxy + hosted dashboard), built on Cloudflare Workers + Supabase.

> Observability tools *show* you spend. FuseGuard *stops* it. Building in public 🛠️
